Cargar movimientos — Pichincha

Varios movimientos del banco de una sola vez. Tipo DE = depósito (entra plata), ND = nota de débito (sale plata). El saldo del banco se actualiza igual que cargando de a uno. Si una línea con datos falla, no se carga ninguna.

Notas: solo Pichincha. El importe va siempre positivo — el tipo (DE/ND) decide si suma o resta del saldo. El Doc. banco es el comprobante del banco (opcional). Las líneas vacías se ignoran.
