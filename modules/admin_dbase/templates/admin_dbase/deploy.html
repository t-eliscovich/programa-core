¿Por qué hay que reiniciar? Flask carga el código Python en memoria una sola vez cuando arranca. git pull sólo cambia los archivos en disco — el proceso vivo sigue sirviendo el código viejo hasta que muere y vuelve a nacer. Por eso son dos pasos: pull (1) trae el código nuevo; reiniciar (2) hace que la app lo cargue.

Paso 1 · git pull origin main

Trae el último commit. Muestra qué archivos cambiaron. No reinicia nada.

(click "Pull" para correr git pull origin main)

Paso 2 · Reiniciar app

Reinicia la Scheduled Task ProgramaCoreApp. La conexión se va a cortar — esperá ~10s y refrescá.

(click "Reiniciar app" después de que el pull termine OK)
